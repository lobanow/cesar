Программа предназначена для шифровки и дешифровки текста шифром цезаря, со смещением необходимиым пользователю

Для запуска программы требуется среда python, нажать ПКМ, открыть с помощью python

Для шифровки/дешифровки
1.Ввести текст латиницей
2.нажать Enter
3.Ввести ключ, на которую программа будет смещать шифр
4.нажать Enter
после этого программа выведет текст со смещением на ключ в обе стороны

Для завершения работы программы можно либо закрыть окно программы, либо нажать Enter после вывода зашифрованного текста
